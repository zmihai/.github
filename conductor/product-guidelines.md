## Naming Conventions
- **Kebab-case:** All workflow filenames, action directory names, and input/output parameters MUST use **kebab-case** (e.g., `reusable-ci.yml`, `setup-node-env`, `node-version`). This ensures consistency with GitHub Actions' native conventions and improves readability across different environments.
